# GPU/CUDA optional requirements for ntsCuda
# Install in addition to requirements.txt to enable GPU acceleration.
#
# CuPy provides a NumPy-compatible GPU array library.
# Choose the package that matches your CUDA toolkit version:
#
#   CUDA 12.x:
#     pip install cupy-cuda12x
#
#   CUDA 11.x:
#     pip install cupy-cuda11x
#
#   CUDA 10.2:
#     pip install cupy-cuda102
#
# To check your CUDA version: nvcc --version  OR  nvidia-smi
#
# If no GPU is present, ntsCuda falls back to CPU mode automatically.
# GPU acceleration benefits:
#   - FFT ringing operations (ringing / ringing2): 5-20x speedup
#   - Color space conversions (bgr2yiq / yiq2bgr): 2-5x speedup
#
# Example full install with GPU support:
#   pip install -r requirements.txt
#   pip install cupy-cuda12x
