# ============================================================
#  GPU acceleration (optional — install ONE cupy wheel)
# ============================================================
# CuPy enables FFT / colour-space GPU speedups in app/ntsc.py.
# Choose the wheel matching your CUDA toolkit:
#
#   CUDA 12.x:   pip install cupy-cuda12x
#   CUDA 11.x:   pip install cupy-cuda11x
#   CUDA 10.2:   pip install cupy-cuda102
#
# Check your CUDA version with:  nvcc --version   (or nvidia-smi)
# No GPU? ntsCuda falls back to CPU mode automatically.

# ============================================================
#  Full CUDA pipeline (optional — app/pipeline_cuda.py)
# ============================================================
# Enables NVDEC decode, Triton-fused kernel, CUDA graphs and
# NVENC encode. Needs all three packages below installed.
#
#   Linux (CUDA 12):
#     pip install torch --index-url https://download.pytorch.org/whl/cu121
#     pip install triton av
#
#   Windows (CUDA 12):
#     pip install torch --index-url https://download.pytorch.org/whl/cu121
#     pip install triton-windows av
#
# If any of these are missing the pipeline logs a warning and
# transparently falls back to the DefaultRenderer path.

# ============================================================
#  macOS Apple Silicon (M1/M2/M3) pinned set
